    Section 5.3 Symmetric and Orthogonal Matrices and Quadratic Forms

    This section needs a lot of work
    🔗

    Definition 5.3.1.

    A square matrix \(A\) is said to be symmetric if \(A=A^{\intercal}\text{.}\)
    🔗
    🔗

    Theorem 5.3.2.

    The eigenvalues of a symmetric matrix are real.
    🔗
    🔗

    Proof.

    Let \(\lambda\) be an eigenvalue of \(A\) with corresponding eigenvector \(\boldsymbol{v}\) .
    🔗
    \begin{equation*} A \boldsymbol{v} = \lambda \boldsymbol{v} \end{equation*}
    🔗
    Left multiply by \(\boldsymbol{v}^{\intercal}\)
    🔗
    \begin{equation*} \boldsymbol{v}^{\intercal} A \boldsymbol{v} = \lambda \boldsymbol{v}^{\intercal} \boldsymbol{v} \end{equation*}
    🔗
    Take the complex conjugate of the eigenvector equation above:
    🔗
    \begin{align*} \overline{\boldsymbol{v}}^{\intercal} A^{\intercal} \amp = \overline{\lambda} \overline {\boldsymbol{v}}^{\intercal} \\ \overline{\boldsymbol{v}}^{\intercal} A \amp = \overline{\lambda} \overline {\boldsymbol{v}}^{\intercal} \end{align*}
    🔗
    left mulitply by \(\boldsymbol{v}\)
    🔗
    \begin{align*} \overline{\boldsymbol{v}}^{\intercal} A \boldsymbol{v} \amp = \overline{\lambda} \overline{\boldsymbol{v}}^{\intercal} \boldsymbol{v} \\ \overline{\boldsymbol{v}}^{\intercal} \lambda \boldsymbol{v} \amp = \overline{\lambda} \overline{\boldsymbol{v}}^{\intercal} \boldsymbol{v} \\ \lambda \overline{\boldsymbol{v}}^{\intercal} \boldsymbol{v} \amp = \overline{\lambda} \overline{\boldsymbol{v}}^{\intercal} \boldsymbol{v} \end{align*}
    🔗
    so \(\lambda = \overline{\lambda}\text{,}\) hence it is real.
    🔗
    🔗

    Theorem 5.3.3.

    The eigenvectors of a symmetric matrix are orthogonal.
    🔗
    🔗

    Definition 5.3.4.

    A square matrix \(Q=[\boldsymbol{q}_1\;\; \boldsymbol{q}_2\;\;\boldsymbol{q}_3\;\; \cdots\;\;\boldsymbol{q}_n] \) is said to be orthogonal if
    🔗
    \begin{equation*} \boldsymbol{q}_i^{\intercal} \boldsymbol{q}_j = 0 \end{equation*}
    🔗
    for all \(i \neq j\text{.}\) If in addition that \(\boldsymbol{q}_i^{\intercal}\boldsymbol{q}_i = 1\) for all \(i\) then the matrix is also said to be orthonormal.
    🔗
    🔗

    Theorem 5.3.5.

    If \(Q\) is a orthonormal matrix, then \(Q^{\intercal} = Q^{-1}\text{.}\)
    🔗
    🔗

    Subsection 5.3.1 Orthogonal Transformations

    Definition 5.3.6.

    Let \(Q\) be a linear transformation from \(\mathbb{R}^n\) to \(\mathbb{R}^n\text{.}\) If \(Q\) is an orthonormal matrix, then the linear transformation is said to be an orthogonal transformation.
    🔗
    🔗

    Example 5.3.7.

    Show that the rotational transformation given in section 7.10 is an orthogonal transformation.
    🔗
    Solution.
    Show that
    🔗
    \begin{equation*} Q = \begin{bmatrix} \cos \theta \amp -\sin \theta \\ \sin \theta \amp \cos \theta \end{bmatrix} \end{equation*}
    🔗
    satisfies \(Q^{\intercal} = Q^{-1}\text{.}\)
    🔗
    🔗
    🔗

    Definition 5.3.8.

    A quadratic form is a polynomial function in \(x_1, x_2, \ldots, x_n\) such that
    🔗
    \begin{equation*} q(\boldsymbol{x}) = a_1 x_1^2 + a_2 x_2^2 + \cdots a_n x_n^2 + b_{12} x_1 x_2 + b_{13} x_1 x_3 + \cdots \end{equation*}
    🔗
    🔗
    Note that a quadratic form can have squared terms and products between only two variables.
    🔗

    Example 5.3.9.

    The following are examples of quadratic forms.
    🔗
    • \(\displaystyle 4x_1^2 + 7x_2^2 + 4x_1x_2\)
      🔗
      🔗
    • \(\displaystyle x_1^2 + 4x_2^2 + 9 x_3^2 -2x_1x_2 + 3x_1x_3 + x_2x_3 \)
      🔗
      🔗
    🔗
    And the following are not:
    🔗
    • \(x_1^2 + 4x_2^2 + 9x_2+ 3\text{.}\)
      🔗
      🔗
    • \(x_1^2 + 4x_2^2 + 3x_3^2 + 2x_1x_2 -3x_1 x_2 x_3 \text{.}\)
      🔗
      🔗
    🔗
    🔗
    A quadratic form can be written as
    🔗
    \begin{equation*} q(\boldsymbol{x}) = \boldsymbol{x}^{\intercal} A \boldsymbol{x} \end{equation*}
    🔗
    for some symmetric matrix \(A\) and \(\boldsymbol{x}=[x_1\;\;x_2 \;\; \cdots \;\; x_n]^{\intercal}\text{.}\)
    🔗

    Example 5.3.10.

    Find the matrix \(A\) for the quadratic forms above:
    🔗
    Solution.
    \begin{align*} A \amp= \begin{bmatrix} 4 \amp2 \\ 2 \amp 7 \end{bmatrix}\\ \boldsymbol{x}^{\intercal} A \boldsymbol{x} \amp = [x_1 \;\; x_2] \begin{bmatrix} 4 \amp2 \\ 2 \amp 7 \end{bmatrix} \begin{bmatrix} x_1 \\ x_2 \end{bmatrix}\\ \amp = [x_1 \;\; x_2] \begin{bmatrix} 4x_1 + 2 x_2 \\ 2 x_1 + 7 x_2 \end{bmatrix}\\ \amp = 4x_1^2 + 2 x_1 x_2 + 2 x_1 x_2 + 7 x_2^2 \end{align*}
    🔗
    And for the second quadratic form:
    🔗
    \begin{equation*} A = \begin{bmatrix} 1 \amp -1 \amp 3/2 \\ -1 \amp 4 \amp 1/2 \\ 3/2 \amp 1/2 \amp 9 \end{bmatrix} \end{equation*}
    🔗
    🔗
    🔗
    🔗

    Subsection 5.3.2 Principle Axes of Quadratic Forms

    For the quadratic form:
    🔗
    \begin{equation*} q(\boldsymbol{x}) = \boldsymbol{x}^{\intercal} A \boldsymbol{x} \end{equation*}
    🔗
    if \(A = P D P^{-1}\) can be written:
    🔗
    \begin{equation*} q(\boldsymbol{x}) = \boldsymbol{x}^{\intercal} P D P^{-1} \boldsymbol{x} \end{equation*}
    🔗
    and since \(A\) is symmetric then the eigenvectors form an orthonormal basis so \(P^{-1} = P^{\intercal}\text{.}\)
    🔗
    \begin{equation*} q(\boldsymbol{x}) =\boldsymbol{y}^{\intercal} D \boldsymbol{y} \end{equation*}
    🔗
    where \(\boldsymbol{y}= P^{\intercal} \boldsymbol{x}\text{.}\) Example of ellipse/contour plot.
    🔗
