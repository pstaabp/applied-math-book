Example 1.3.17.
Write down the solution set of the following linear system given in the reduced row echelon form of the matrix:
\begin{equation*} \left[\begin{array}{rrrrrr|r} 1 \amp 0 \amp 3 \amp 0 \amp 0 \amp 5 \amp 11 \\ 0 \amp 1 \amp -4 \amp 0 \amp 0 \amp 9 \amp -4 \\ 0 \amp 0 \amp 0 \amp 1 \amp 0 \amp 3 \amp 5 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp 0 \amp -2 \\ \end{array}\right] \end{equation*}
If we add a row between the 2nd and 3rd rows as well as after the fourth row, the matrix is:
\begin{equation*} \left[\begin{array}{rrrrrr|r} 1 \amp 0 \amp 3 \amp 0 \amp 0 \amp 5 \amp 11 \\ 0 \amp 1 \amp -4 \amp 0 \amp 0 \amp 9 \amp -4 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \\ 0 \amp 0 \amp 0 \amp 1 \amp 0 \amp 3 \amp 5 \\ 0 \amp 0 \amp 0 \amp 0 \amp 1 \amp 0 \amp -2 \\ 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \amp 0 \end{array}\right] \end{equation*}
The the solution set is
\begin{equation*} \left\{ \begin{bmatrix} 11 \\ -4 \\ 0 \\ 5 \\ -2 \\ 0 \end{bmatrix} + \begin{bmatrix} -3 \\ 4 \\ 1 \\ 0 \\ 0 \\ 0 \end{bmatrix} x_3 + \begin{bmatrix} -5 \\ -9 \\ 0 \\ -3 \\ 0 \\ 1 \end{bmatrix} x_6 \; | \; x_3, x_6 \in \mathbb{R} \right\} \end{equation*}
Recalling that you need to replace the 0 in the 3rd row of the \(x_3\) vector with a 1 and replace the 0 in the 6th row of the \(x_6\) vector with a 1.
