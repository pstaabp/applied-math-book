Example 1.3.24. Solve a Linear System using matrices.

Solve the following linear system use matrices and Gaussâ€™ method.
\begin{align*} x-2y + 4z \amp = -15 \\ -3x +0y + 2z \amp = -7 \\ x-2y + 4z \amp = -15 \end{align*}
Solution.
First write the linear system as a matrix
\begin{align} \left[\begin{array}{rrr|r} -1 \amp -4 \amp 10 \amp -37 \\ -3 \amp 0 \amp 2 \amp -7 \\ 1 \amp -2 \amp 4 \amp -15 \\ \end{array} \right] \notag\\ \begin{array}{r} -3 R_1 + R_2 \rightarrow R_2 \\ R_1 + R_3 \rightarrow R_3, \end{array} \qquad \left[\begin{array}{rrr|r} -1 \amp -4 \amp 10 \amp -37 \\ 0 \amp 12 \amp -28 \amp 104 \\ 0 \amp -6 \amp 14 \amp -52 \end{array}\right] \notag\\ R_2 +2R_3 \rightarrow R_3, \qquad \left[\begin{array}{rrr|r} -1 \amp -4 \amp 10 \amp -37 \\ 0 \amp 12 \amp -28 \amp 104 \\ 0 \amp 0 \amp 0 \amp 0 \end{array}\right] \tag{1.3.3} \end{align}
The matrix in ((1.3.3)) is in echelon form, so we use back substitution to solve for \(x_1\) and \(x_2\) in terms of \(x_3\text{,}\) the free variable. The 2nd row of ((1.3.3)) can be written:
\begin{equation*} 12 x_2 = 104 + 28 x_3 \end{equation*}
or
\begin{equation*} x_2 = 17 + \frac{7}{3} x_3 \end{equation*}
and using this in the first equation of ((1.3.3))
\begin{equation*} -x_1 - 4(17+\frac{7}{3} x_3) + 10 x_3 = -37 \end{equation*}
and solving for \(x_1\)
\begin{equation*} x_1 = -31+\frac{2}{3} x_3 \end{equation*}
