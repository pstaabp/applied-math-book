Solution 3.3.2.1.

First, the set \(\mathbb{R}^n\) is a vector space. Next, we need to show that the three properties are satisfied for \(\langle \vec{u}, \vec{v} \rangle = \vec{u}^T \vec{v}\text{.}\)
  1. \begin{align*} \langle \vec{u}, \vec{v} \rangle \amp = \vec{u}^T \vec{v} \end{align*}
    since the transpose of a number is itself
    \begin{align*} \amp = (\vec{u}^T \vec{v})^T \end{align*}
    using the properties of transposes in SubsectionÂ 2.1.4.
    \begin{align*} \amp = \vec{v}^T (\vec{u}^T)^T = \vec{v}^T \vec{u} = \langle \vec{v}, \vec{u} \rangle. \end{align*}
    so the first property is satisfied.
  2. \begin{align*} \langle a \vec{u} + b \vec{v}, \vec{w} \rangle \amp = (a \vec{u} + b\vec{v})^T \vec{w} = (a \vec{u}^T + b \vec{v}^T) \vec{w} \\ \amp = a \vec{u}^T \vec{w} + b \vec{v}^T \vec{w} = a \langle \vec{u}, \vec{w} \rangle + b \langle \vec{v}, \vec{w} \rangle. \end{align*}
  3. \begin{equation*} \langle \vec{u}, \vec{u} \rangle = \vec{u}^T \vec{u} = u_1^2 + u_2^2 + u_3^2 + \cdots + u_n^2 \end{equation*}
    which satisfies \(\geq 0\text{.}\) The only time that this quantity equals 0, is when \(u_1=u_2 = \cdots =u_n\) or \(\vec{u} = \vec{0}\text{.}\)
