Theorem 3.5.5.

Given a nonzero vector \(\vec{v}\) and a vector \(\vec{u}\) both in \(\mathbb{R}^n\text{,}\) the projection map in ((3.5.2)) is a linear transformation.

Proof.

We need to prove the two properties of linear transformations in DefinitionÂ 3.4.1.
\begin{align*} \text{Proj}_{\vec{v}}(\vec{u} + \vec{w}) \amp = \frac{\langle (\vec{u}+\vec{w}), \vec{v} \rangle}{\langle \vec{v},\vec{v} \rangle} \\ \amp = \frac{\langle \vec{u},\vec{v} \rangle + \langle \vec{w},\vec{v} \rangle}{\langle \vec{v},\vec{v} \rangle } \\ \amp = \frac{\langle \vec{u},\vec{v} \rangle}{\langle \vec{v},\vec{v} \rangle } + \frac{\langle \vec{w},\vec{v} \rangle}{\langle \vec{v},\vec{v} \rangle } \\ \amp = \text{Proj}_{\vec{v}} \vec{u} + \text{Proj}_{\vec{v}} \vec{w} \end{align*}
and
\begin{align*} \text{Proj}_{\vec{v}} (\alpha \vec{u}) \amp = \frac{\langle (\alpha \vec{u}), \vec{v} \rangle}{\langle \vec{v},\vec{v} \rangle} \\ \amp = \alpha \frac{\langle \vec{u}, \vec{v} \rangle}{\langle \vec{v},\vec{v} \rangle} = \alpha \text{Proj}_{\vec{v}} \vec{u} \end{align*}
