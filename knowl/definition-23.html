Definition 2.1.1.

In this case, let \(k \in \mathbb{R}\) and \(A\) and \(B\) are \(m \times n\) matrices. Then
  1. \(kA\) is the matrix formed by each entry is \(ka_{i,j}\text{.}\) That is multiply each entry by \(k\text{.}\)
  2. \(A+B\) is the matrix form by each entry is \(a_{i,j}+b_{i,j}\text{.}\) That is to add matrices, we add element by element.
  3. \(A-B\) is the matrix form by each entry is \(a_{i,j}-b_{i,j}\text{.}\) That is to add matrices, we subtract element by element.
If \(A\) and \(B\) are not the same size then \(A+B\) and \(A-B\) does not exist.
