Proof

\(\Longrightarrow\) Let \(||\vec{u}-\vec{v}||=0\text{,}\) then \(\langle \vec{u}-\vec{v},\vec{u}-\vec{v} \rangle=0\) By definition, this is only zero if \(\vec{u}-\vec{v}=\vec{0}\text{,}\) therefore \(\vec{u}=\vec{v}\text{.}\)
\(\Longleftarrow\) If \(\vec{u}=\vec{v}\text{,}\) then \(||\vec{u}-\vec{v}|| = ||\vec{0}||=0\text{.}\)
