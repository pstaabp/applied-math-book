Definition 6.1.7.

A function \(f\) has a left-hand derivative at \(x=x_0\) if
\begin{equation*} f'(x_0^-) = \lim_{h \rightarrow 0^-} \frac{f(x_0+h)-f(x)}{h} \end{equation*}
exists. Similarly, a function \(f\) has a right-hand derivative at \(x=x_0\) if
\begin{equation*} f'(x_0^+) = \lim_{h \rightarrow 0^+} \frac{f(x_0+h)-f(x)}{h} \end{equation*}
exists.
