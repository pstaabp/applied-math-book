Item 3.
Recall that for a matrix to be nonsingular, that a linear system with coefficients from the matrix has a unique system or the row reduced echelon form of the matrix has no row of zeros. Let \(T\) be the original matrix and \(\hat{T}\) is the matrix in echelon form. If the \(\hat{T}\) is nonsingular, then it is the identity matrix, which has the nonzero determinant 1.
A singular matrix \(T\) reduces to a matrix \(\hat{T}\) with a row of zeros. By #2, this determinant is 0.
