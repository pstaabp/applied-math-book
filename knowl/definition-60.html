Definition 6.1.6.

A function \(f\) has a left-hand limit at \(x=x_0\) if
\begin{equation*} f(x_0^-) = \lim_{x \rightarrow x_0^-} f(x) \end{equation*}
exists. In addition, a function \(f\) has a right-hand limit at \(x=x_0\) if
\begin{equation*} f(x_0^+) = \lim_{x \rightarrow x_0^-} f(x) \end{equation*}
exists.
