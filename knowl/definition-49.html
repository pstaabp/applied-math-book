Definition 3.3.7.

Let \(\{\vec{v}_1, \vec{v}_2, \ldots, \vec{v}_n\}\) each be elements of an inner product space, \(V\text{.}\) The set is called an orthonormal set if
\begin{equation*} \langle \vec{v}_i, \vec{v}_j \rangle = 0 \qquad \text{if $i \neq j$} \end{equation*}
and
\begin{equation*} ||\vec{v}_i|| = \sqrt{\langle \vec{v}_i, \vec{v}_i \rangle} = 1 \end{equation*}
If only the first condition holds, the set is called orthogonal.
