Example 2.6.19.

Show that the Rank-Nullity Theorem holds for Examples Example 2.6.15 and Example 2.6.17.
Solution.
In Example Example 2.6.15, the \(2 \times 3\) matrix is reduced to reduced row echelon form and it shows that there are 2 non-zero rows, hence the rank of the matrix is 2. Also in Example Example 2.6.15, the nullity was shown to be 1 and the sum is 3, the number of columns of \(A\text{.}\)
In Example Example 2.6.17, the \(4 \times 4\) matrix is reduced to reduced row echelon form and it shows that there are 2 non-zero rows, and again the rank of the matrix is 2. The example also shows that the nullity is 2 and the sum is 4, the number of columns of \(A\text{.}\)
